# scripts/

> "Please excuse the crudity of this model. I didn't have time to build it to scale or to paint it." — Dr. Emmett Brown

These scripts are for **example/prototype purposes only**. They demonstrate using the HCP Terraform API to create and update workspaces programmatically. They are not production-ready and should not be treated as such.

## Scripts

- **create_workspace.py** — Creates a VCS-integrated workspace, sets initial variables, and triggers/applies a run.
- **update_workspace.py** — Updates an existing workspace to enable KV and triggers/applies a new run.
